# Misión Admisión OS

Misión Admisión OS es el portal interno de acceso a herramientas, procesos y recursos clave de la operación de Misión Admisión.

Su objetivo es centralizar accesos, ordenar recursos por área, facilitar el onboarding y reducir la dependencia de preguntar dónde está cada herramienta o documento.

## Objetivo del MVP

Construir una primera versión navegable del portal con:

- login simple de acceso
- home principal con identidad visual espacial
- áreas principales del sistema
- accesos rápidos
- catálogo inicial de recursos

## Áreas iniciales

- Operaciones

- General

## Principios del proyecto

- No duplicar información innecesariamente
- Usar el portal como capa ordenadora
- Crecer por fases
- Priorizar utilidad operativa sobre complejidad técnica
- Mantener una experiencia clara, limpia y escalable

## Roadmap inicial

### Fase 1
- Base visual del portal
- Hero/banner principal
- Sección de accesos rápidos
- Tarjetas por área
- Catálogo inicial cargado desde JSON

### Fase 2
- Vista de recursos por área
- Mejoras UX/UI

### Fase 3
- Login real
- Roles y visibilidad por área
- Administración de contenido
